## Run Locally

**Prerequisites:**  Node.js

1. Node.js must be installed before installing dependencies:
use this link to download node.js:
`https://nodejs.org/en/download`


2. Install dependencies:
   `npm install`

3. Run the app:
   `npm run dev`

4. Currently running my personal api key but if you want your own api key go to `Micro-Shift-main\services\geminiService.ts`
